How to get this? (Show instructions)

Steps to get the Bearer Token:

  1. Open praamid.ee in a new tab and log in if you aren't already.
  2. Open your browser's Developer Tools (usually by pressing F12).
  3. Go to the "Network" tab in the Developer Tools.
  4. Filter by "Fetch/XHR".
  5. On the praamid.ee page, select a direction and change the date to trigger a schedule load.
  6. Find a request to /online/events?... in the Network tab. Click on it.
  7. Under the "Headers" (or "Request Headers") section for that request, find the Authorization header.
  8. The value will look like Bearer eyJhbGciOiJSUzI1NiI.... Copy the entire string starting with "Bearer ".
  9. Paste it into the input field above.
